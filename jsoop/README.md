## 面向对象与面向过程编程
面向过程，其实就是主要是以``动作``为核心，而面向对象是以``事物``为核心，举个形象的栗子。
> 面向过程： 吃(人,饭)
面向对象：人.吃（饭）

## 继承
一般面向对象语言，都有两种继承方式：接口继承和实现继承。EMCAScript在ES5之前，只支持实现继承。ES6开始支持extend
### 原型链继承
子.prototype = new 父
缺点：
1. 无法实现多继承
2. 引用属性复用
### 构造继承
在子的构造函数中，通过```父.call(this)```，继承父。
